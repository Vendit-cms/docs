## 상품 생성

![상품\_3.png](/images/상품_3.png)

<Check>
  상품 생성 입력 정보

  - **객실 타입 (\*필수)**
    - 상품과 연결할 객실 타입을 선택해 주세요. 재고는 해당 객실 타입의 객실 수를 기준으로 설정돼요.
    - 한 번 생성된 상품의 객실 타입은 변경할 수 없어요.
  - **객실 타입의 운영 중지 객실이 왜 재고에 포함되나요?**

    운영 중지 객실도 재고에 포함되는 이유는, 공사나 점검 등으로 일시적으로 운영을 멈췄다가 다시 활성화할 때 기존 재고 설정을 그대로 유지하기 위해서예요.

    상품 재고는 \*\*객실 타입의 전체 객실 수(운영 중지 객실 포함)\*\*를 기준으로 설정돼요. 하지만 실제 판매 관리에서는 **운영 중인 객실 수만**을 기준으로 재고가 판매돼서, 운영 중지된 객실이 판매되지 않도록 안전하게 관리돼요.
  - **상품 이름 (\*필수)**
  - **기본 통화**
    - 상품 요금에 적용할 기본 통화를 설정해 주세요.
    - 한 번 생성된 상품의 통화는 변경할 수 없어요.
  - **판매 시간표**
    - 상품의 입실 시간과 퇴실 시간을 설정해요.
  - **재고 세부 설정**
    - **최대 판매 재고** : 판매 가능한 수량을 제한할 수 있어요. 객실 타입의 객실 수까지 설정할 수 있어요.
    - **재고 자동 보충** : 연결된 모든 채널 상품의 잔여 수량을 보충해 재고를 일정하게 유지해요. 설정된 수량이 실제 재고와 같으면, 판매 시 함께 차감돼요.
</Check>

<Note>
  **객실 타입의 운영 중지 객실이 왜 재고에 포함되나요?**

  - 운영 중지 객실도 재고에 포함되는 이유는, 공사나 점검 등으로 일시적으로 운영을 멈췄다가 다시 활성화할 때 기존 재고 설정을 그대로 유지하기 위해서예요.
  - 상품 재고는 \*\*객실 타입의 전체 객실 수(운영 중지 객실 포함)\*\*를 기준으로 설정돼요. 하지만 실제 판매 관리에서는 운영 중인 객실 수만을 기준으로 재고가 판매돼서, 운영 중지된 객실이 판매되지 않도록 안전하게 관리돼요.
</Note>

### 인원별 요금

![상품\_9.png](/images/상품_9.png)

<Check>
  인원별 요금 설정을 지원하는 채널에서는, 투숙객이 기준 인원을 초과하면 객실 요금에 1인당 추가 요금이 적용돼요.

  - 인원별 요금 설정을 지원하지 않는 채널은 최대 인원 수를 기준으로 요금이 책정돼요.
</Check>
