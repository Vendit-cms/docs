---

### 숙박업소 생성

![숙박업소 생성\_1.png](/images/숙박업소생성_1.png)

- 숙박업소를 검색하면, 숙박업소 이름 / 주소 / 숙박 유형(모텔, 호텔, 펜션, 게스트하우스)이 자동으로 입력돼요.
  - 신규 업장으로 숙박업소를 찾을 수 없는 경우, 정보를 직접 입력해 주세요.

![숙박업소 생성\_2.png](/images/숙박업소생성_2.png)

- **이름**
  - 숙박업소 이름을 입력해요.
- **주소**
  - 숙박업소 주소 정보를 선택하거나 직접 입력해요.
- **숙박업소 유형**
  - 숙박업소의 유형을 분류해요.
  - 주소와 숙박 유형 정보는 서비스 기능에는 영향을 주지 않으며, 벤디트 내부에서 숙박업소를 분류하는 용도로만 사용돼요.
- **기본 통화**
  - 숙박업소에서 활용되는 기본 통화로 상품과 요금 생성 시 기본 통화로 활용됩니다.
- **기본 입** · **퇴실 시간**
  - 예약 정보에 입 · 퇴실 시간이 없는 판매 채널과의 연동 시 기본 연동 시간으로 활용돼요.
