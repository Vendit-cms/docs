### 예약 상태

<Check>
  예약 상태 종류

  - **전체** : 모든 상태(완료, 취소, 배정, 대기)의 예약 내역 확인
  - **완료** : 연동된 예약
  - **취소** : 취소된 예약
  - **배정** : 예약을 연동 서비스(VCLOUD)에서 객실에 배정한 상태
  - **대기** : 예약 연동되지 않고 대기 중인 상태
</Check>

### 예약 검색

<Check>
  예약자, 예약 번호, 전화번호를 검색하여 특정 예약을 빠르게 조회할 수 있어요.
</Check>

### 화면 보기 설정

<Check>
  화면에 표시할 항목을 선택하고 보이는 순서는 원하는데로 변경할 수 있어요.
</Check>

### 기본 화면 보기 항목

<Check>
  - **예약 확인** : 예약 정보 확인 여부
  - **이용 유형** : 예약 이용 유형(ex. 숙박, 대실)
  - **예약 상태** : 예약의 상태(완료, 취소, 배정, 대기)
  - **입실 일시** : 입실 예정 일시
  - **퇴실 일시** : 퇴실 예정 일시
  - **박 수** : 체류 기간
  - **총 인원** : 예약된 투숙 인원(정보값 없을 경우 기본값 1명)
  - **예약처** : 연동된 예약 출처
  - **연동 서비스** : 연결된 연동 서비스
  - **예약자** : 예약 투숙객 이름
  - **객실 타입** : 연동된 예약의 객실 타입
  - **예약 번호** : 연동된 예약의 채널 예약 번호
  - **예약 수량** : 연동된 예약의 수량
  - **전화번호** : 연동된 예약의 투숙객 전화번호
  - **판매 금액** : 연동된 채널의 판매가
  - **메모** : 숙박업소에서 기록한 메모 표시, 예약 상세에서 수정할 수 있어요.
  - **연동 일시** : 예약이 연동된 일시
</Check>

### 기본 화면 숨김 항목

<Check>
  - **요청** : 예약 시 투숙객이 작성한 요청 사항
  - **상품** : 채널 상품과 연결된 VCMS 상품 정보
  - **채널 상품** : 연동 예약의 채널 상품
  - **입금 금액** : 연동된 채널의 입금가
  - **미수 금액** : 연동된 채널의 미수금
  - **이동 수단** : 연동된 예약의 이동 수단
  - **취소 일시** : 연동된 예약의 취소 일시
  - **성인, 아동, 유아** : 예약 시 연령별 인원 수
</Check>

<Check>
  여기어때, 네이버, 야놀자는 예약 정보에 입실 인원 정보가 없어서 표기되지 않아요.
</Check>
