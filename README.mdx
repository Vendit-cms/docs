# VCMS 사용자 가이드

Vendit Channel Manager(VCMS) 문서에 오신 것을 환영합니다.

본 가이드는 다음과 같은 내용을 제공합니다:

- 숙박업소 설정 및 운영 기본 절차
- 요금 / 재고 구성 및 패키지 설정
- 판매 채널 연동 및 동기화 방식
- 예약 내역 조회 및 관리 방법
- 기준 정보(이용 약관, 개인정보 처리방침 등)

좌측 메뉴에서 원하는 항목을 선택해 주세요.
